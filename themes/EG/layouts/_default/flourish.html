Created as a traditional LAMP stack project, the site is built on a foundation of Silverstripe CMS with custom page templates.

Custom page templates allow the External Relations team the ability to create content with no technical experience. Design elements are individual, reusable components that use standard HTML, CSS & Javascript and used across multiple co-brands for consistency.

To facilitate collection of user forms, "User Defined Forms" module augmented Silverstripe core functionality. CLC has specific requirements around form auto completion, contributions to the open source project were made to add required functionality. Contributions to the project were archived by Github into the Arctic Code Vault in 2020.

Integrations were created to add additional third party data dynamically on the site using API provided by each integrator. News with infinite scrolling, social aggregation and rss support was integrated using Flockler. Custom filterable calendar to show events from the College managed information system iSams. Templates to pull staff vacancies from HR system using Vacancy filler API.

Initially locally hosted at the College, the project was migrated to AWS for improved resilience and scale. AWS additions included edge caching to improve the experience for overseas users with AWS Cloudfront, alongside security enhancements from AWS Shield.
