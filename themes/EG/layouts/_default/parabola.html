Parabola Arts

parabolaarts.cheltladiescollege.org

Co-Brand microsite for The Cheltenham Ladies' College revolving around the theatre and arts centre in the centre of Cheltenham.

The site allowed visitors to book tickets for upcoming events using an integration with spetrix booking and ticketing system, allowing visitors to choose their seat and see a view from that location before buying a ticket.

Created as a traditional LAMP stack project, the site is built on a foundation of Silverstripe CMS with custom page templates, the site extended the existing Cheltenham Ladies' College site adding additional page templates.

Page templates allow the venues team the ability to create content with no technical experience, events created automatically populated the ticketing system so visitors could purchase tickets via the website.

The website also allowed visitors to check in for events on the day and provided access to an e-ticket. Spetrix was used as a third party ticketing system integration.

Initially locally hosted at the College, the project was migrated to AWS for improved resilience and scale. AWS additions included edge caching to improve the experience for overseas users with AWS Cloudfront, alongside security enhancements from AWS Shield.
