The Cheltenham Ladies' College are one of the oldest and most prestigious schools in the UK. A namesake of distinctive, high-end single sex education offering young women pioneering opportunities.

Reflected in an identity that feels elevated across both printed and digital brand touchpoints balancing the traditional and minimal. The branding defines the view of modern education through sophisticated imagery as a glimpse into life at CLC.

The identity is flexible enough to act as a versatile backdrop for the variety of co-brands and internal services.

Created as a traditional LAMP stack project, the site is built on a foundation of Silverstripe CMS with custom page templates.

Custom page templates allow the External Relations team the ability to create content without technical experience. Components that use standard HTML, CSS & Javascript can be reused across multiple co-brands for consistency.

To facilitate collection of user data, "User Defined Forms" module augmented Silverstripe core functionality. CLC has specific requirements around form auto completion, contributions to the open source project were made to add required functionality. Contributions to the project were archived by Github into the Arctic Code Vault in 2020.

Integrations were created to add additional third party data dynamically on the site, using API provided by each integrator. News with infinite scrolling, social aggregation and rss support using Flockler. Custom filterable calendar to show events from the College managed information system iSams. Templates to pull staff vacancies from HR system using Vacancy filler API.

Initially locally hosted at the College data centre, the project was migrated to AWS for improved resilience and scale. AWS additions included edge caching to improve the experience for overseas users with AWS Cloudfront, alongside security enhancements from AWS WAF & Shield.

Progressive enhancements to the site have been added over the life of the project improving accessability support, service workers as well as new functionality to enhance user engagement with the addition of pop-out notices.
