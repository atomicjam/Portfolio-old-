Created as a traditional LAMP stack project, the site is built on a foundation of Silverstripe CMS with custom page templates.

Custom page templates allow the Guild team the ability to create content with no technical experience. Design elements are individual, reusable components that use standard HTML, CSS & Javascript and used across multiple co-brands for consistency.

Integrations for Blackbaud allowed the Guild community to create an account and share information with the wider community.

Initially locally hosted at the College, the project was migrated to AWS for improved resilience and scale. AWS additions included edge caching to improve the experience for overseas users with AWS Cloudfront, alongside security enhancements from AWS Shield.

Progressive enhancements to the site have been added over the life of the project improving accessability support, service workers as well as new functionality to enhance user engagement with the addition of pop-out notices.
